# 데모 실행하는 방법

```bash
$ npm start
```

# 참고해야하는 파일

config.ts  
routes/index.ts  
lib/live-streaming-api.ts  

# Host가 영상 스트리밍 시작하는법

1. Connect 버튼 클릭
2. Get Media 버튼 클릭
3. 미디어 송출 되는 것 확인
4. Cast Media 버튼 클릭

주의사항 : 웹페이지가 https 보안이 되어있지 않으면 getUserMedia 안됨. localhost는 괜찮음.
